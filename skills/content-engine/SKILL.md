# Content Engine

Gerar conteúdo adaptado por plataforma a partir de um brief único. Não é o mesmo texto redimensionado.

## When to Activate

- Produzir conteúdo pra múltiplas plataformas
- Adaptar artigo longo pra posts, threads, emails
- Criar variações pra testar ângulos
- Produzir em escala com consistência de voz

## Pre-Writing Checklist

Before drafting, confirm:
1. **Platforms** — quais? (X, LinkedIn, email, blog, Instagram)
2. **Voice** — qual perfil de voz? (verificar se existe em `brand-voice` skill)
3. **Goal** — engajamento, conversão, educação, awareness?
4. **Audience** — quem lê?
5. **Brief** — tema, ângulo, tom, referências

## Platform Specs

| Platform | Char Limit | Format | Tone | Key Rule |
|----------|-----------|--------|------|----------|
| X/Twitter | 280 | Short, punchy | Direct, witty | Lead with hook |
| LinkedIn | 3,000 | Professional post | Thought leadership | Story + insight |
| Email | — | Subject + body | Personal, clear | Subject line = 50% of success |
| Blog | — | Long-form, SEO | Authoritative | H2 every 300 words |
| Instagram | 2,200 | Caption + hashtags | Visual-first | First line = hook |

## Workflow

```
1. CONFIRM  → platforms, voice, goal, audience
2. BRIEF    → extract key message per platform
3. DRAFT    → write each platform version (different! not copy-paste)
4. VERIFY   → check each: char limit, format, tone, no AI patterns
5. DELIVER  → present all versions with rationale
```

## Voice Profile Application

If a voice profile exists:
- Read it first
- Apply tone, vocabulary, sentence structure
- Don't just "use the brand voice" — embody it

If no voice profile:
- Ask user for tone reference
- Create one for future use

## Quality Gates

| Gate | Check |
|------|-------|
| Platform-native | Each version genuinely different (not copy-paste) |
| Char limits | Within platform constraints |
| No AI patterns | No "game-changer", "leverage", "seamless" |
| Hook quality | First line grabs attention |
| CTA clarity | Clear next step for reader |

## Anti-patterns

- ❌ Same text with minor tweaks per platform
- ❌ AI vocabulary in any version
- ❌ Generic introductions ("In today's world...")
- ❌ No clear CTA or purpose
- ❌ Exceeding char limits

## Output

Present as:
```
## [Platform Name]
[Content]

**Rationale:** [Why this approach for this platform]
```
