### Problema Inicial
- OpenClaw não conseguia enviar comandos para o Antigravity
- `agy chat "prompt"` abria a janela mas não enviava o texto

### Solução Desenvolvida
- Script PowerShell usa simulação de teclado:
  1. Abre painel com `agy chat --reuse-window`
  2. Espera 4s para UI carregar
  3. Usa **Ctrl+Alt+Shift+C** para focar no input
  4. SendKeys digita o prompt
  5. ENTER final envia
