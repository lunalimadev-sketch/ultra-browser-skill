# Tech Stack - Luna Agency

Este arquivo contém as configurações técnicas, preferências de modelos e integrações do ambiente de trabalho.

## Preferências de Modelos
- **Principais:** Hunter Alpha (1T), Gemini 2.0 Flash, Kimi K2.5.

## Ambiente
- **OS:** Windows (PowerShell).
- **Shell:** PowerShell.

## Integrações Configuradas
- **Google Workspace CLI (gws):** Instalado e configurado globalmente.
- **ClickUp:** Integrado via scripts de PowerShell no workspace.
- **Telegram Bridge:** Controle duplo e injeção de prompts do Antigravity via Telegram Bot ([Detalhes](telegram_integration.md)).
